# graph_rs

## Usage

To graph some demo equations:
```
cargo run -- --demo
```


To enter custom equations:
```
cargo run
```
- exponents must be integers

Can be compiled to Web Assembly and accessed in a browser at localhost:8000.
```
./serve.sh
```
